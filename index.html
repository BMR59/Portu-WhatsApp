Welcome to the PortuWhatsApp Homepage

This is the homepage of the PortuWhatsApp project for the University of Pittsburgh's Computational Methods in the Humanities course. This project analyzes the conversations between native Portuguese speakers and native English Portuguese learners through the texting app "WhatsApp." You can read more about the project and how the data was collected on the about page.
